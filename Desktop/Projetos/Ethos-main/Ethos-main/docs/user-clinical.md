# Guia de Uso Clínico - ETHOS

## Fluxo de Trabalho Ético
1. **Consentimento:** Sempre obtenha o consentimento do paciente antes de gravar ou importar áudios.
2. **Rascunho:** O prontuário gerado automaticamente é sempre um **Rascunho**. Ele deve ser revisado.
3. **Validação:** Após a revisão, clique em "Validar Prontuário" para assinar o documento. Uma vez validado, ele se torna imutável para garantir a integridade clínica.

## Privacidade
- Todos os dados são armazenados localmente.
- Nenhuma informação clínica sai do seu computador.
- Você pode realizar o expurgo total dos dados nas Configurações.

## Requisitos
- CPU com suporte a AVX2 (recomendado).
- Mínimo 8GB de RAM.
- Espaço em disco para os modelos de IA (~3GB).
